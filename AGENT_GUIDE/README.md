# AGENT_GUIDE - Guia de Desenvolvimento

Este diretório contém toda a documentação técnica necessária para desenvolver e manter o projeto Ethos Financeiro.

## 📚 Documentação

- **01_CODING_PATTERNS.md** - Padrões de código a seguir
- **02_PROTECTED_FILES.md** - Arquivos que não devem ser modificados
- **03_PROBLEMS_SOLUTIONS.md** - Problemas e soluções encontrados
- **04_DEVELOPMENT_GUIDE.md** - Guia completo de desenvolvimento
- **05_PYWEBVIEW_GUIDE.md** - Guia específico de pywebview
- **06_ISSUES_DB.md** - Banco de dados de problemas ativos
- **07_DIAGNOSTIC_REPORT.md** - Relatório de diagnóstico do aplicativo
- **08_ERROR_LOG.md** - Registro de erros e soluções

## 🎯 Como Usar

1. **Antes de iniciar desenvolvimento:**
   - Leia `04_DEVELOPMENT_GUIDE.md`
   - Confirme arquivos não são afetados
   - Verifique problemas ativos em `06_ISSUES_DB.md`

2. **Durante desenvolvimento:**
   - Siga padrões de `01_CODING_PATTERNS.md`
   - Documente novos problemas em `03_PROBLEMS_SOLUTIONS.md`
   - Atualize `06_ISSUES_DB.md` quando resolver problemas

3. **Ao terminar desenvolvimento:**
   - Verifique se arquivos protegidos não foram alterados
   - Teste todas as funcionalidades
   - Atualize documentação se necessário

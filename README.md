<h2> Desenvolvimento <h2><br>

Crie um ambiente virtual e ative.

`python -m venv venv`

Linux:
  
`source venv/bin/activate`
  
Windows:
  
Powershell: `PS \Scripts\Activate.ps1`
  
CMD: `\Scripts\activate.bat`
  
Qualquer duvida sobre os comandos acessar o link: https://docs.python.org/pt-br/3/library/venv.html

Caso o Windows esteja dando erro de permissão.

<p>Execute como administrador: `Set-ExecutionPolicy Unrestricted`</p>
<br><p>Agora instale as dependências.</p>

`pip install -r requirements.txt`
